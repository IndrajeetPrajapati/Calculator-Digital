# Calculator Digital

A modern and responsive digital calculator built using **HTML, CSS, and JavaScript**. Designed to perform basic arithmetic operations with a clean user interface and smooth button interactions.

## 🧮 Features

* Addition, Subtraction, Multiplication, Division
* Clear / Reset Function
* Backspace Delete Button
* Responsive Design for Mobile and Desktop
* Smooth Button Click Effects
* Clean Digital Display UI
* Fast and Accurate Calculations

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

## 📌 Purpose

This project was created to practice JavaScript logic, DOM manipulation, event handling, and responsive frontend design.

## 🚀 How to Use

1. Click number buttons.
2. Choose arithmetic operators.
3. Press "=" to get result.
4. Use "C" to clear all values.
